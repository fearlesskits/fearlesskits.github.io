> **Wear the Legacy, Be Fearless!**

A premium showcase website for Fearless Kits — your go-to destination for authentic imported player edition jerseys.

## 🚀 Deploy to GitHub Pages (Free!)

### Step 1: Create a GitHub Account
1. Go to [github.com](https://github.com) and sign up (if you haven't already)

### Step 2: Create a New Repository
1. Click the **+** button → **New repository**
2. Name it: `fearless-kits` (or `fearless-kits.github.io` for the cleanest URL)
3. Set it to **Public**
4. Click **Create repository**

### Step 3: Upload Your Files
1. On the repository page, click **"uploading an existing file"**
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `images/` folder (with all jersey images)
3. Click **Commit changes**

### Step 4: Enable GitHub Pages
1. Go to your repository → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select **main** branch → **/ (root)**
4. Click **Save**

### Step 5: Your site is live! 🎉
Your website will be available at:
- `https://YOUR-USERNAME.github.io/fearless-kits/`
- Or `https://YOUR-USERNAME.github.io/` (if you named the repo `YOUR-USERNAME.github.io`)

## 📱 Social Media Links
Update these links in `index.html`:
- Facebook Page: Search for `#social-facebook` and update `href`
- Instagram: Search for `#social-instagram` and update `href`
- TikTok: Search for `#social-tiktok` and update `href`

## 🎨 Customization
- **Colors**: Edit CSS variables in `style.css` under `:root`
- **Products**: Edit the `products` array in `script.js`
- **Images**: Replace images in the `images/` folder
- **Prices**: Update prices in the `products` array in `script.js`
